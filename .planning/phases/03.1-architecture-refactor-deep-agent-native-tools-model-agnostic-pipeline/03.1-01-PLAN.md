  <action>
Rewrite each tool module. The pattern for each is the same: rename `_xxx_impl` to the public name, remove the `@tool` decorator and `StructuredTool` alias at the bottom, remove the `from langchain_core.tools import tool` import. Each module should export a plain function that `create_deep_agent` can accept as a callable.

**calculate.py:**
- Rename `_calculate_impl` -> `calculate`, `_pct_change_impl` -> `pct_change`, `_sum_values_impl` -> `sum_values`
- Remove the 3 lines at the bottom: `calculate = tool("calculate")(_calculate_impl)` etc.
- Remove `from langchain_core.tools import tool`
- Keep `Optional[str]` on pct_change unit_old/unit_new parameters (per research: Pydantic V2 still needs this)
- Keep all docstrings — they serve as tool descriptions for create_deep_agent auto-discovery
- Keep the `_eval_node` private helper and all other internal functions unchanged

**route_files.py:**
- Rename `_route_files_impl` -> `route_files` (the public function)
- Delete the `_route_files_agent` thin wrapper entirely
- Delete the `route_files = tool("route_files")(...)` alias at the bottom
- Remove `from langchain_core.tools import tool`
- Remove the `Config` parameter from `route_files`. Instead, add a module-level constant:
  `_CORPUS_DIR = Path(os.environ.get("CORPUS_DIR", str(Path("../corpus/transformed").resolve())))`
- Inside `route_files(question: str) -> dict`, use `_CORPUS_DIR` instead of `config.corpus_dir`
- Remove the `TYPE_CHECKING` import block and `from src.config import Config` entirely
- Keep `extract_years`, `fy_to_calendar_months`, `year_to_months` as-is (they have no _impl pattern)

**search_in_file.py:**
- Rename `_search_in_file_impl` -> `search_in_file`
- Remove `search_in_file = tool("search_in_file")(...)` alias
- Remove `from langchain_core.tools import tool`
- Keep `normalize_query`, `build_spans` public helper functions unchanged

**extract_table_block.py:**
- Rename `_extract_table_block_impl` -> `extract_table_block`
- Remove `extract_table_block = tool("extract_table_block")(...)` alias
- Remove `from langchain_core.tools import tool`
- Keep `_collect_table_block` private helper unchanged

**normalize_answer.py:**
- Rename `_normalize_answer_impl` -> `normalize_answer`
- Remove `normalize_answer = tool("normalize_answer")(...)` alias
- Remove `from langchain_core.tools import tool`

For ALL modules: preserve every docstring (they become tool descriptions), preserve all type annotations, preserve all internal logic. This is a rename + cleanup, not a logic change.
