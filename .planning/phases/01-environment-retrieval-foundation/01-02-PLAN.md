1. Create `workspace/src/tools/__init__.py` (empty).

2. Create `workspace/src/tools/route_files.py`:

   **Core function:** `route_files(question: str, config=None) -> dict`

   Returns a dict with either:
   - `{"paths": [list of absolute path strings], "years_found": [...], "fy_mapped": bool}`
   - `{"error": "no_year_found", "question": question}` when no year can be extracted

   **Year extraction (regex, no LLM):**
   - Pattern 1: "FY YYYY" or "FY{YYYY}" or "fiscal year YYYY" (case-insensitive) -> fiscal year
   - Pattern 2: "FY YY" or "FY{YY}" (two-digit) -> expand to four-digit: YY >= 30 maps to 19YY, YY < 30 maps to 20YY (covers the 1939-2025 corpus range)
   - Pattern 3: Bare four-digit year (1939-2025 range) -> calendar year
   - Pattern 4: "calendar year YYYY" -> calendar year (explicit)
   - Extract ALL year references found (a question may mention multiple years)

   **Fiscal year to calendar month mapping:**
   - US fiscal year ends September 30
   - FY YYYY = October (YYYY-1) through September (YYYY)
   - `fy_to_calendar_months(fy_year: int) -> list[tuple[int, int]]`:
     Returns list of (year, month) tuples: [(YYYY-1, 10), (YYYY-1, 11), (YYYY-1, 12), (YYYY, 1), ..., (YYYY, 9)]

   **Calendar year mapping:**
   - Calendar year YYYY -> all 12 months of YYYY: [(YYYY, 1), ..., (YYYY, 12)]

   **Path construction:**
   - For each (year, month) pair, construct filename: `treasury_bulletin_{year}_{month:02d}.txt`
   - Resolve to absolute path using `config.corpus_dir / filename`
   - CRITICAL: validate each path exists. Use the set of actual files in corpus_dir (call `os.listdir` or use manifest). Only include paths that exist.
   - If none of the constructed paths exist, still return the paths list (empty) with a note — do NOT hallucinate alternative filenames.

   **Result limiting:**
   - If more than 12 candidate files found (e.g., full FY + calendar year both extracted), keep the FY files first (more specific), then calendar year files, capped at 12.
   - For a typical single-year question, expect 1-12 files.

   **Important edge cases:**
   - Questions mentioning a single year like "1940" may refer to a bulletin published in 1941 (data about 1940 appears in January 1941 bulletin). This is handled by the BM25 search in Plan 03, NOT here. The file router returns all bulletins from the mentioned year.
   - Questions with no year at all: return the error dict.

   **Helper functions (module-level, exported for testing):**
   - `extract_years(question: str) -> list[dict]`: returns `[{"year": int, "type": "fiscal"|"calendar"}]`
   - `fy_to_calendar_months(fy_year: int) -> list[tuple[int, int]]`
   - `year_to_months(year: int) -> list[tuple[int, int]]`: calendar year -> 12 months
  </action>
