## Decisions Made

- **a2a-sdk uses snake_case Pydantic fields:** AgentCard fields are `default_input_modes` and `default_output_modes`, not camelCase. Discovered by inspecting `AgentCard.model_fields` after SDK install.
- **build(routes=[...]) works:** A2AStarletteApplication.build() passes **kwargs to Starlette constructor — passing `routes=[Route("/health", ...)]` adds the extra route without any manual app manipulation.
- **context.get_user_input() preferred over manual part parsing:** RequestContext provides a `get_user_input()` helper that handles TextPart structure correctly. Replaces the `_get_message_text()` helper from the plan template.
- **[http-server] extra required:** Base a2a-sdk install raises ImportError for sse-starlette. Changed requirements entry to `a2a-sdk[http-server]>=0.3.20,<1.0`.
- **Two Python environments on machine:** pip uses anaconda Python 3.11; PATH Python is 3.12. All verification uses `/c/Users/mdado/anaconda3/python.exe` explicitly.
