**Part A: Create retrieval counter-wrappers (workspace/src/tools/retrieval_wrappers.py)**

Create `workspace/src/tools/retrieval_wrappers.py` with factory functions that wrap the raw retrieval tools with per-run call counters. Do NOT use ToolCallLimitMiddleware — use this counter-wrapper approach instead.

```python
from langchain_core.tools import tool
from src.tools.route_files import route_files as _raw_route_files
from src.tools.search_in_file import search_in_file as _raw_search_in_file

def make_counted_route_files(limit: int = 20):
    """Create a @tool-decorated route_files that returns RETRIEVAL_EXHAUSTED after `limit` calls."""
    call_count = {"n": 0}  # mutable container for closure

    @tool(name="route_files")
    def counted_route_files(question: str) -> dict:
        """Route a question to the relevant corpus files. Returns file paths matching the question's date range and topic."""
        call_count["n"] += 1
        if call_count["n"] > limit:
            return {"error": "RETRIEVAL_EXHAUSTED"}
        # Delegate to the raw tool's underlying function
        return _raw_route_files.func(question) if hasattr(_raw_route_files, 'func') else _raw_route_files(question)

    return counted_route_files

def make_counted_search_in_file(limit: int = 20):
    """Create a @tool-decorated search_in_file that returns RETRIEVAL_EXHAUSTED after `limit` calls."""
    call_count = {"n": 0}

    @tool(name="search_in_file")
    def counted_search_in_file(file_path: str, query: str) -> dict:
        """Search within a specific file for spans matching the query. Returns ranked text spans."""
        call_count["n"] += 1
        if call_count["n"] > limit:
            return {"error": "RETRIEVAL_EXHAUSTED"}
        return _raw_search_in_file.func(file_path, query) if hasattr(_raw_search_in_file, 'func') else _raw_search_in_file(file_path, query)

    return counted_search_in_file
```

IMPORTANT: The docstrings on counted_route_files and counted_search_in_file must match the original tool descriptions closely enough for the LLM to use them correctly. Copy/adapt the original docstrings.

IMPORTANT: The counter resets naturally because `make_counted_*` creates a fresh closure each time it is called. `run_question` calls `create_agent` fresh per question, which calls `make_counted_*` fresh, so the counter resets per question. This satisfies AGT-02 idempotency.

IMPORTANT: The wrapper MUST call the underlying function, not the StructuredTool object with positional args. After Plan 01 decorates the raw tools with `@tool`, `_raw_route_files` will be a `StructuredTool`. Access the underlying function via `.func` attribute, or use `.invoke()` with a dict. Choose whichever approach works — test it.
